# Main GIS Software

## GIS Platforms


| Software                | Install Method                                                                             | What it does / Why I need it                                                 |
| ----------------------- | ------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------- |
| **Google Earth Engine** | [Online](https://earthengine.google.com/)                                                  | Cloud platform for large-scale raster processing and analysis.               |
| **ArcGIS Pro**          | [Manual (D)](https://pro.arcgis.com/en/pro-app/latest/get-started/download-arcgis-pro.htm) | Primary vector data work plus secondary raster analysis; final maps/layouts. |
| **QGIS**                | [UniGetUI (D)](https://www.qgis.org/download/)                                             | Open-source GIS with a large plugin/community tool ecosystem.                |
