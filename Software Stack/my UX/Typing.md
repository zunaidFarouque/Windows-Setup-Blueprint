# Typing

I use WhisperTyping because of its generous free tier.
Another option that automatically fixes the text to proper formatting is typeless (installed with UniGetUI)

# Clipboard

I use copyQ for clipboard. Read the doc here: [CopyQ](../../docs/CopyQ/copyq.md)

# DIFF: Compare and merge

I use winmerge. Settings are:

- **Compare > General**
  - Enable moved block detection: On
  - Align similar lines: On
  - Diff algorithm: `minimal`
  - Enable indent heuristic: On
- **Editor > Compare/Merge**
  - Copy granularity for selected differences: `Char`
  - View line differences: On
  - Line difference coloring: Character-level
