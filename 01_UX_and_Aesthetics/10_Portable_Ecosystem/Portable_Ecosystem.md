## Quick Start

- [`Copy_Shortcuts_to_StartMenu.ps1`](Copy_Shortcuts_to_StartMenu.ps1): Copies shortcut files from `D:\_installed\_Shortcuts` into a dedicated Start Menu folder (`My Portable Apps`) so portable apps show up like normal installed apps. It also clears Explorer's icon cache and restarts Explorer so shortcut icons refresh properly.
- [`Portable_Apps_with_Scoop_UniGetUI.ubundle`](Portable_Apps_with_Scoop_UniGetUI.ubundle): UniGetUI export bundle for this portable ecosystem. Useful as a restore/import snapshot for the Scoop and Winget app set managed through the `scoop_UGU` workflow.
