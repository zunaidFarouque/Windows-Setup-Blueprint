# Foobar2000 as a tray / deskband player

**Date:** 2018-06-08

## Purpose

Use Foobar2000 instead of a dedicated tray player (e.g. @Max Tray Player): richer features with similar RAM use, plus optional taskbar hiding so the main window does not stay on the taskbar.

## Goals

- Tray-friendly playback with solid customization.
- Low RAM footprint (~7 MB in this setup).
- Player can be hidden from the taskbar when not needed.

## Setup

### 1. Install Foobar2000

Install from the official installer (or your usual archive, e.g. Google Drive mirror).

### 2. Install components

| Component                                                                                     | Role                               |
| --------------------------------------------------------------------------------------------- | ---------------------------------- |
| [**foo_deskband_controls**](https://www.foobar2000.org/components/view/foo_deskband_controls) | Toolbar / deskband control surface |

Install via _File → Preferences → Components_ (or place DLLs in the user components folder as usual).

### 3. Configure foo_deskband_controls

Adjust the deskband layout in the component's settings, or install the saved preset from the repo for a quick start.

How to install the preset:

1. Copy [`Deskband_presets/_2o21_final.json`](Deskband_presets/_2o21_final.json) (my preset file).
2. Paste it into:

```text
C:\Users\USERNAME\AppData\Roaming\DeskbandControls
```

3. In `foo_deskband_controls`, load that preset from the preset directory if it does not appear automatically.
4. If needed, update the Foobar2000 path inside the preset to match your actual install path.

---

After that, run Foobar2000 and enable the deskband from the taskbar toolbar menu as needed for your Windows version.
