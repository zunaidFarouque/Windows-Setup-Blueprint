## Run services

Open in **mp3DirectCut**:

Download from here: https://mpesch3.de/

```
"D:\_installed\_productivity\Foobar 2.1.3 32bit\mp3DirectCut\mp3DirectCut.exe"
```

**Audacity**:

Should be installed as portable, using scoop_UGU.com launcher

```
D:\_installed\audacity-win-3.7.5-64bit\Audacity.exe "%_path%"
```

**Song Master Pro**:

Paid software for practicing songs, seperating stems (remove voice).

```
"D:\_installed\Song Master Pro 2.7\SongMaster Pro.exe" "%_path%"
```

**Antares Auto-Key**:

Find scale/key of music.

```
"C:\Program Files\Common Files\VST3\Antares\Auto-Key.exe"
```

**KeyFinder**:

Find key of music, open source, DL:

- (win exe) https://www.oldergeeks.com/downloads/file.php?id=4609
- (official website) https://www.ibrahimshaath.co.uk/keyfinder/

```
"D:\_installed\_productivity\Foobar 2.1.3 32bit\KeyFinder-WIN\KeyFinder.exe"
```
